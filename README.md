# Node & Mongoose Starter

Node and Mongoose Basic Environment

1. install node, yarn, and MongoDB
2. yarn install
3. mongod
4. yarn server

# Execute

1. yarn test (test mode)
2. yarn api (production server)
3. yarn server (development mode)
